

# git 账号设置
```
1. git config --global user.name "hijj"
2. git config --global user.email "wangjiafight@gmail.com"
3. ssh-keygen -C 'wangjiafight@gmail.com' -t rsa
```

# git 远程库推送
```
1. git init
2. git remote add origin set-url https://my_token@github.com/hijj/notebook.git
3. git push --set-upstream origin master

```
